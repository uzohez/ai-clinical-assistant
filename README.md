# 🏥 AI Clinical Assistant

An AI-powered clinical decision support system that performs:

- Symptom intake
- Intelligent cross-examination
- Risk classification
- Clinical recommendations

## 🚀 Features

- Conversational AI (multi-turn)
- Persistent memory (SQLite)
- Risk level detection
- Structured medical outputs

## 🧠 Tech Stack

- FastAPI
- OpenAI API
- SQLite
- JavaScript (frontend)

## ⚠️ Disclaimer

This system provides AI-assisted insights only and is not a medical diagnosis tool.

```bash
pip install -r requirements.txt
